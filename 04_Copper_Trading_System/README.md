一、策略說明（銅交易策略）
交易標的：銅
時間週期：日線
__
📈 做多（買進）
•    MA20 向上突破 MA60
•    RSI(14) > 55
📉 做空（賣出）
•    MA20 向下跌破 MA60
•    RSI(14) < 45
⛔ 出場

二、策略概述
•    使用 yfinance 獲取銅期貨資料
•    以 50 日 / 200 日移動平均線 判斷趨勢
•    使用 RSI 作為動能與過熱判斷
•    結合趨勢與動能，降低假突破風險
三、主要功能
📊 數據獲取
•    取得 2020–2025 銅期貨價格資料
🧮 數據預處理
•    標準化數據
•    建立 60 天時間序列
🤖 LSTM 模型
•    雙層 LSTM + Dropout
•    Optimizer：Adam
•    訓練 50 epochs
•    驗證資料佔 10%
📐 評估指標
•    MSE
•    MAE
•    RMSE
🔮 未來預測
•    根據最後序列預測未來 30 天銅價
