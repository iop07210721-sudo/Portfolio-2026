# 📈 銅期貨交易預測工具
**Copper Futures Trading Forecasting Toolm**

## 專案簡介
本專案為「程式交易 / 量化分析」相關課程與自主研究成果，系統以 銅期貨（Copper Futures, HG=F） 為研究標的，整合 Python 量化分析 與 GitHub Actions 雲端自動化 技術，目標在於建立一套 可回測、可監控、可擴充 的商品期貨量化交易架構。
銅作為重要的工業金屬，其價格高度反映全球製造業、能源轉型與經濟循環狀態，本系統提供具量化依據的趨勢判斷與進出場訊號，降低主觀判斷風險。

## 🚀 核心功能
本系統採用 Hybrid 混合式架構，區分為「趨勢決策支援模組」與「策略研究與參數測試模組」兩種運作模式。

### 1. 📊 趨勢決策支援模組（Decision Support Module）
🔹 不直接下單
 而是提供：
🔹多 / 空 / 觀望訊號
🔹趨勢延續或反轉判斷
🔹工業金屬景氣解讀依據

### 2. ☁️ 策略研究與參數測試模組（Strategy Research）
🔹 不同均線參數比較
🔹 不同 RSI 閾值測試
🔹 策略虧損原因分析
